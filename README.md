# mi-nomina

Aplicación frontend simple para generación de nominas / cuentas de cobro.

## Descripción
Proyecto creado con Vite + React (JSX) y TailwindCSS que contiene componentes y utilidades para generar cuentas de cobro.

## Estructura
- `index.html` - punto de entrada HTML
- `src/` - código fuente
  - `App.jsx` - componente principal
  - `main.jsx` - bootstrap de la app
  - `index.css` - estilos globales
  - `pages/Usuario.jsx` - ejemplo de página
  - `utils/generarCuentaCobro.js` - utilidades

## Requisitos
- Node.js 16+ recomendado
- npm

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Build
```bash
npm run build
```
